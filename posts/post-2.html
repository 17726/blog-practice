掌握用于现代网页设计的CSS Grid布局

作者：17726 · 2025-07-18

CSS Grid布局是一个强大的工具，可以轻松创建复杂、响应式的网页布局。与主要用于一维布局的Flexbox不同，Grid擅长于二维排列。

基本概念

CSS Grid的核心是容器（设置了 display: grid 的元素）和项目（容器的子元素）。您在容器上定义列和行，然后可以将项目放置在此网格中。


.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 1rem;
}

这个简单的示例创建了一个三列等宽的网格，并在它们之间设置了间隙。从这里开始，可能性是无穷的。
